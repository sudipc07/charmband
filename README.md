# CHARM Band Website

Static marketing site for CHARM™ Band, designed for Cloudflare Pages.

## Deploy

Cloudflare Pages can serve this repo with no build step.

- Production branch: `main`
- Build command: none
- Output directory: `/`

Initial deployment URL: `https://charmband.pages.dev`

## Local Preview

```sh
python3 -m http.server 8080
```

Then open `http://127.0.0.1:8080`.
